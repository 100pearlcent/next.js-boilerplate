## ✉️ Commit Convention

|  | Usage          | Same As       | 
|  :---:   | ------------- | ------------- | 
| 🏁  | 프로젝트 생성 | `Init`|
| 🔧  | config 파일  | `Config`|
| ➕ | 의존성 추가  | `Add`|
| ➖  | 의존성 제거  | `Remove`|
| ⚙️  | 상수, 유틸 함수 | |
| 💾  | 정적 파일, 데이터 | | 
| 🧱    | 컴포넌트  | `Feat` |
| 📄    | 페이지  | `Feat` |
| 🥚  | 추상화  |   |
| 🖼️    | UI 레이아웃  |  |
| 📡    | API 연결  | |
| 🩹   | 버그 픽스  |  `Fix` |
| 🚒  | 핫픽스  |  `HotFix` |
| 🗑️    | 불필요한 소스 제거 | `Remove` |
| 🧹   | 오타, 공백 수정  |  `Chore` |
